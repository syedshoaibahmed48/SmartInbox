# SmartInbox

Securely interact with your emails through a simple AI chat interface. 

---

## Overview
Smart Inbox displays your emails and provides an interactive chat interface where you can:
- Ask questions about your inbox.
- Summarize messages.
- Interact with your email data privately and securely.

---

## Tech Stack
- Nextjs: Frontend & Proxy
- Fast API: Backend
- Redis: temporary storage
- llama.cpp: llm inference

---

## Focus Areas
- Data privacy and confidentiality
- Secure authentication and token handling
- Minimal data retention and encryption-first design
